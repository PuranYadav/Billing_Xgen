# Billing_Xgen
 Customer,Invoice, Payments management


Billing Xgen is a complete PHP & MySQL based billing and invoice management system designed for small and medium businesses.

FEATURES:- Secure Login System Customer Management Invoice Creation & Management Auto GST Calculation (CGST / SGST / IGST) Payment Management Module Outstanding Balance Tracking Invoice Status (Paid / Partial / Unpaid) PDF Invoice Generation Dashboard Analytics Search & Date Filters Responsive Bootstrap UI

TECHNOLOGY:- PHP MySQL Bootstrap 5 JavaScript / jQuery

MODULES:- Customer Management Add Customer Edit Customer Customer Search Date-wise Filtering Invoice Management Create Invoice Edit Invoice Auto Calculation Engine Payment Management Receive Payments Payment History Balance Tracking Outstanding Management

DASHBOARD:- Total Customers Total Invoices Total Revenue Payment Summary Business Statistics
